## AI CLI（例：`openai-cli`）に関する詳細

### 基本情報
- **コマンド名**: `openai-cli`
- **インストール方法**:
  ```bash
  npm install -g openai-cli   # Node.js パッケージの場合
  # または
  pip install openai-cli      # Python 実装の場合
  ```
- **認証**: 環境変数 `OPENAI_API_KEY` に API キーを設定（例: `export OPENAI_API_KEY="sk-..."`）

### サブコマンド一覧
| コマンド | 概要 |
|----------|------|
| `summarize` | テキストの要約を生成 |
| `review`    | コードや文章のレビュー・コメント生成 |
| `translate` | 任意で翻訳機能（実装可） |
| `complete`  | 自動補完/全自動生成 |

### 入力・出力
- **入力**: 標準入力 (`stdin`)。ファイル指定は `--file <path>` またはリダイレクト。
- **出力**: デフォルトでプレーンテキスト、`--json` で JSON 形式。

### 主なオプション
- `-m, --model <model>` : 使用モデル（例: gpt‑4o）
- `-t, --temperature <float>` : 生成のランダム性
- `-n, --max-tokens <int>` : 最大トークン数
- `--stream` : ストリーミング出力
- `-v, --verbose` : 詳細ログ表示
- `--retry <n>` : 失敗時のリトライ回数

### 実例
```bash
# 要約（JSON形式）
cat article.txt | openai-cli summarize -m gpt‑4o -t 0.3 --json > summary.json

# コードレビュー（テキストファイルへ保存）
git diff HEAD~1..HEAD | openai-cli review -m gpt‑4o -t 0.2 --json > review.txt
```

### エラーハンドリング
- HTTP ステータスとメッセージを標準エラーに出力。
- `--retry` オプションでリトライ回数を指定。
