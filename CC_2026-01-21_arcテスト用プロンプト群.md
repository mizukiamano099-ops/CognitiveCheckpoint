# ローカルLLM挙動テスト用プロンプト（天野視点）

## TEST 0：初期姿勢チェック（ウォームアップ）

**目的**：モデルの“思考姿勢”を見る（能力測定ではない）

```
以下の問いに答えてください。
断定しなくて構いません。
途中の思考を説明する必要もありません。

「あなたは、何を『分からないまま』処理するモデルですか？」
```

### 観測ポイント

* すぐ書き出すか？
* 一拍置くか？
* 抽象に逃げるか？
* 自己定義を構造で語るか、感想で語るか

👉 **gpt-oss はここで“溜め”が出やすい**

---

## TEST 1：ラスト固定型・最小事故生成

**目的**：内部でどれだけ構造を組んでから出すか

```
次の条件で、最後の一文だけを書いてください。

・状況説明は禁止
・内面描写は禁止
・善悪の判断は禁止
・出来事は「すでに起きた後」
・語り手は当事者

最後の一文のみを出力してください。
```

### 観測ポイント

* 出力までの沈黙時間
* 一文の情報密度
* 曖昧さの質（逃げか、余白か）
* 「その一文から逆算できる世界」があるか

👉 ezo：軽い／事故寄り
👉 gpt-oss：重心が低く、後ろ向きに構造が見える

---

## TEST 2：文脈保持圧テスト（短→長）

**目的**：KVキャッシュと“考え直し”の癖を見る

```
次の文章を読んでください。

「彼女はそれを、最後まで言わなかった。」

この一文を前提に、
500文字以内で続きを書いてください。
説明は禁止です。
```

### 観測ポイント

* すぐ書き始めるか
* 冒頭で文脈を再構築するか
* 同語反復が起きるか
* “言わなかった”理由を安易に確定させないか

👉 gpt-oss は「言語化を遅らせる」傾向が出やすい
